# 💼 PeopleCore - Front-end

Sistema ERP moderno focado em **Gestão de Pessoas**, desenvolvido para centralizar e otimizar processos administrativos relacionados a colaboradores, departamentos e organização empresarial.

O **PeopleCore** foi construído com foco em experiência do usuário, organização escalável e interface profissional para ambientes corporativos modernos.

---

# 💙 Sobre o Projeto

O **PeopleCore** é uma aplicação Front-end desenvolvida para simular um sistema ERP voltado à gestão empresarial e recursos humanos, oferecendo uma experiência intuitiva, moderna e eficiente para administração de pessoas dentro de organizações.

Este repositório contém toda a estrutura visual e funcional do Front-end da aplicação, construída utilizando conceitos modernos de desenvolvimento web.

A aplicação foi projetada com foco em:

- 👥 Gestão inteligente de colaboradores
- 🏢 Organização de departamentos
- 🎨 Interface moderna e profissional
- 📱 Responsividade completa
- ♻️ Componentização reutilizável
- 🚀 Performance e escalabilidade
- 🧠 Navegação intuitiva
- 🔐 Estrutura preparada para autenticação e integrações futuras

# 🚀 Como Executar o Projeto

## 📥 Clone o repositório

```bash
git clone https://github.com/seu-repositorio/peoplecore-front.git
```

---

## 📂 Acesse a pasta do projeto

```bash
cd peoplecore-front
```

---

## 📦 Instale as dependências

```bash
npm install
```

---

## ▶️ Execute o projeto

```bash
npm run dev
```

---

# 🌐 Integração com o Back-end

O Front-end foi estruturado para consumir uma API REST responsável pelo gerenciamento de:

- Funcionários
- Departamentos
- Autenticação
- Processos administrativos

A arquitetura foi planejada para permitir operações completas de CRUD com escalabilidade e organização empresarial.
